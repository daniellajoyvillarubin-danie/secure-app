# secure-app

## Advanced DevSecOps Features

This project includes:

- Static Application Security Testing (SAST) using Semgrep
- Vulnerability scanning using Trivy
- Docker container scanning
- Software Bill of Materials (SBOM) generation
- Trusted build promotion
- GitHub Actions artifact uploads
- Multi-job CI pipeline with dependency enforcement

Artifacts generated:
- SBOM (CycloneDX)
- Vulnerability scan reports
- Trusted build metadata
- Container image scan reports
